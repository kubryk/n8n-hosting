# n8n Hosting Configurations

Docker-based hosting configurations for n8n workflow automation platform.

## Setup Options

- **n8n-postgres**: Basic n8n setup with PostgreSQL database
- **n8n-postgres-workers**: n8n with PostgreSQL and worker nodes
- **n8n-workers-cluster**: Worker nodes for distributed setup (deploy on separate servers)

## Quick Start

1. Choose your configuration
2. Navigate to the desired folder
3. Run `docker-compose up -d`
